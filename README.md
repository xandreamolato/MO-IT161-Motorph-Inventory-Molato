# Atelier Luxury — Interactive Website
---

## Project Overview

**Atelier Luxury** is a minimalist fashion and lifestyle website that showcases curated, timeless pieces inspired by craftsmanship and refined living.  
This project demonstrates modern front-end web development using **HTML**, **CSS**, **JavaScript (DOM Manipulation)**, and **Bootstrap 5** to create a responsive and elegant interface.

The website highlights:
- A responsive **Bootstrap-based layout**
- Smooth scrolling and **JavaScript-enhanced interactivity**
- A functional **contact form with validation**
- A refined and consistent design aesthetic aligned with luxury branding principles

---

## Features and Functionalities

### 1. **Responsive Navigation Bar**
- Implemented using Bootstrap’s navbar component  
- Includes a toggle button for mobile view  
- Changes appearance on scroll (using JavaScript DOM manipulation)

### 2. **Hero Section**
- Clean hero banner featuring brand tagline and a call-to-action button  
- Uses Bootstrap grid for alignment between text and imagery  
- Fully responsive across screen sizes

### 3. **Featured Collection**
- Showcases three sample luxury items using Bootstrap **card components**  
- Includes hover animations and smooth visual effects via CSS transitions  
- Button interactivity preserved with DOM event listeners

### 4. **About Section**
- Uses Bootstrap grid and custom CSS to present brand philosophy  
- Includes hover animations for emphasis on values and identity

### 5. **Contact & Inquiries Form**
- Uses Bootstrap form controls for consistent styling  
- **JavaScript validation** for name, email, and message fields  
- Provides user feedback dynamically without reloading the page

### 6. **Smooth Scrolling & User Experience Enhancements**
- Smooth transitions when navigating sections via navbar  
- Dynamic feedback messages with Bootstrap alert components  
- Optimized spacing and typography for premium, elegant feel

## JavaScript Functionality Overview

| Feature | Description |
|----------|-------------|
| **Smooth Scrolling** | Uses `window.scrollTo()` for animated navigation between page sections |
| **Navbar Scroll Effect** | Adds/removes `navbar-scrolled` class when scrolling beyond 50px |
| **Form Validation** | Checks for empty fields and validates email format |
| **Dynamic Feedback** | Displays alert messages (`success`, `warning`, `danger`) using DOM manipulation |
| **Form Reset** | Automatically clears inputs after successful submission |
